# Vedderheia 177

Webpresentasjon for byggeprosjektet på Vedderheia 177.

## Lokal visning

Prosjektet skal lokalt kjøres på fast port `4177`.

Start server fra prosjektmappen:

```bash
python3 -m http.server 4177
```

Åpne deretter:

`http://localhost:4177`

## Struktur

- `index.html` inneholder presentasjonens innhold og seksjonsstruktur.
- `styles.css` inneholder visuell utforming, responsiv oppførsel og print-styles for PDF.
- `script.js` håndterer enkel navigasjon på mindre skjermer.

## Arbeidsnotat

Port `4177` er valgt som fast utviklingsport for dette prosjektet, slik at lokal forhåndsvisning alltid skjer på samme adresse.
